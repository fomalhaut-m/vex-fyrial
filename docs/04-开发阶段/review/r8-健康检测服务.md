### 2.2 ⚠️ 建议改进项

#### 2.2.1 音频播放检测可能产生声音

**问题**：`_checkAudioPlayback()` 中设置音量 `0.01` 后播放 500ms，虽然音量很小，但在安静环境可能听到声音。

```dart
await testPlayer.setVolume(0.01);
await testPlayer.play();
await Future.delayed(const Duration(milliseconds: 500));
await testPlayer.stop();
```

**建议**：可以接受，但建议在测试环境下执行，或者改为只检查加载能力（`setAsset`），不实际播放。

#### 2.2.2 音频焦点检测创建了未使用的AudioPlayer

**问题**：`_checkAudioFocus()` 创建 AudioPlayer 只为测试能否实例化，用完 dispose。逻辑正确但可以简化为直接检查库是否可用。

**建议**：可以接受，当前实现更可靠。

#### 2.2.3 OSS连接检测只检查网络，不检查实际连接

**问题**：只检查 `ConnectivityResult.none`，不实际尝试连接 OSS endpoint。

```dart
if (connectivityResult.contains(ConnectivityResult.none)) {
  // 离线
} else {
  // 在线，但没实际测试 OSS 连接
}
```

**建议**：可以接受，真实的 OSS 连接检测应该在 OSSSync 模块中进行。
