#### 2.1.1 播放器资源释放
```dart
void _dispose() {
  try {
    if (_isInitialized) {
      _player.dispose();  // ✅ 正确释放
      _isInitialized = false;
    }
  }
}

// ref.onDispose 在 build() 中注册
ref.onDispose(() {

});
```

#### 2.1.2 音频服务资源释放
```dart
void dispose() {
  try {
    _playingSubscription?.cancel();   // ✅ StreamSubscription 取消
```
