### 2.2 ⚠️ 建议改进项

#### 2.2.1 缺少搜索页路由

**问题**：PRD要求搜索功能，但路由中没有注册 SearchPage。

```dart
// 根据 PRD：
// /search → SearchPage（Tab 2 内跳转）
```

**建议**：添加搜索页路由：
```dart
GoRoute(
  path: '/search',
  name: 'search',
  builder: (context, state) => const SearchPage(),
),
```

#### 2.2.2 缺少歌单详情页路由

**问题**：PRD要求歌单详情页，但路由中没有注册。

```dart
// 根据 PRD：
// /playlist/:id → PlaylistDetailPage
```

**建议**：添加歌单详情页路由：
```dart
GoRoute(
  path: '/playlist/:id',
  name: 'playlist_detail',
  builder: (context, state) {
    final id = state.pathParameters['id']!;
    return PlaylistDetailPage(playlistId: id);
  },
),
```

#### 2.2.3 缺少统计详情页路由

**问题**：PRD要求统计详情页，但路由中没有注册。

```dart
// 根据 PRD：
// /stats-detail → StatsDetailPage
```
