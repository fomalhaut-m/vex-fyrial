# 产品路线图 - v2

> ⚠️ v2 修订：Phase 1 = 核心（流畅播放+后台保活），统计/歌词补全全降级

---

## Phase 排序原则

**先核心，后增值**
1. 先把播放流畅做好
2. 先把后台保活做好
3. 再做云同步（差异化价值）
4. 最后做增值功能（统计/歌词）

---

## Phase 1：MVP · 流畅播放 ⭐⭐⭐
**目标**：本地音乐能播放、流畅不卡

### 交付功能
- [ ] 本地音乐扫描（指定目录递归扫描）
- [ ] 歌曲列表展示（封面/歌名/歌手/时长）
- [ ] 音频播放（播放/暂停/上一首/下一首/进度拖动）
- [ ] 全屏播放页（封面+播控）
- [ ] 底部迷你播放器（全局常驻）
- [ ] 内存优化（< 150MB）

### 技术重点
- just_audio 播放器集成
- on_audio_query 本地扫描
- 内存占用控制

**验收标准**：播放1万首歌曲不卡，内存占用<150MB

---

## Phase 2：后台保活 ⭐⭐⭐
**目标**：切App/锁屏继续播，通知栏控制

### 交付功能
- [ ] 后台播放（App切后台音乐继续）
- [ ] 通知栏显示（封面/歌名/播控按钮）
- [ ] 锁屏控制（上一首/暂停/下一首）
- [ ] 耳机线控（有线/蓝牙耳机按钮）

### 技术重点
- audio_service 集成
- Android前台Service保活
- iOS后台音频模式
- 华为/小米/OPPO国产ROM适配

**验收标准**：锁屏播放30分钟不掉线，通知栏控制正常

---

## Phase 3：OSS 双向同步 ⭐⭐
**目标**：本地和云端双向同步，跨设备管理

### 交付功能
- [ ] OSS 配置（Bucket/Endpoint/AK/SK）
- [ ] 手动同步（立即同步按钮）
- [ ] 自动同步（定时/增量）
- [ ] 同步状态显示（已连接/同步中/失败）
- [ ] 冲突处理（以时间戳为准）

### 技术重点
- aliyun_oss_flutter_sdk
- flutter_secure_storage（AK/SK安全存储）
- 增量同步算法
- 同步队列管理

**验收标准**：上传100首歌<10分钟，同步成功率>95%

---

## Phase 4：歌词体验 ⭐⭐
**目标**：歌词滚动同步，体验流畅

### 交付功能
- [ ] LRC 文件解析
- [ ] 歌词实时滚动（当前行高亮）
- [ ] 封面/歌词切换（Tab）
- [ ] 点击歌词跳转播放位置
- [ ] 手动滚动打断自动滚动

### 技术重点
- LRC 时间戳正则解析
- ScrollController 联动
- 手动/自动滚动状态切换

**验收标准**：歌词同步准确率>99%（标准LRC文件）

---

## Phase 5：歌单管理 ⭐
**目标**：本地歌单创建和管理

### 交付功能
- [ ] 创建歌单（名称/封面）
- [ ] 添加歌曲到歌单
- [ ] 歌单详情页
- [ ] 删除/重命名歌单
- [ ] 从歌单移除歌曲

### 技术重点
- SQLite 数据库
- PlaylistService CRUD

---

## Phase 6：本地搜索 + 设置 ⭐
**目标**：快速找到歌曲，配置播放参数

### 交付功能
- [ ] 本地歌曲搜索（歌名/歌手/专辑）
- [ ] 搜索历史
- [ ] 音质选择（普通/高品/无损）
- [ ] 播放模式默认（列表循环/单曲/随机）
- [ ] 播放倍速（0.5x ~ 2.0x）

---

## Phase 7：封面歌词自动补全 ⭐
**目标**：自动补全缺失的封面和歌词

### 交付功能
- [ ] 无封面歌曲 → 联网搜索 → 嵌入 ID3 标签
- [ ] 无歌词歌曲 → 联网搜索 LRC → 嵌入/同级保存
- [ ] 匹配错了 → 手动重新匹配/手动选择
- [ ] 可开关（联网功能可关闭）

### 技术重点
- QQ音乐/网易云封面API
- LRC 歌词搜索
- ID3 标签写入（flutter_id3_reader）
- 备份机制（.bak 文件）

---

## Phase 8（可选）：播放统计 ⭐
**目标**：看看听了多少

### 交付功能
- [ ] 每首歌播放次数
- [ ] 每首歌累计播放时长
- [ ] 最后播放时间
- [ ] 按分类统计（流行/摇滚/古典）
- [ ] 按播放时段统计（morning/afternoon/evening/night）

### 注意
此阶段为增值功能，Phase 1-7 已交付可用的音乐播放器后再做

---

## 各阶段依赖关系

```
Phase 1（播放） ──┬── Phase 2（后台保活）
                 │
                 └── Phase 3（OSS同步）─── Phase 4（歌词）
                                               │
                              Phase 5（歌单）──┘
                              Phase 6（搜索）
                              Phase 7（封面歌词补全）
                              Phase 8（统计）
```

---

## 一句话总结

**Phase 1 = 播得动；Phase 2 = 跑得稳；Phase 3 = 同步得了；Phase 4+ = 体验更好**
