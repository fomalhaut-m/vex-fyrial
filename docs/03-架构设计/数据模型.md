# Vexfy 数据模型文档

## 1. SongModel（歌曲模型）

歌曲是核心实体，同时适用于本地音乐和在线音乐。

### 字段定义

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `id` | `String` | 歌曲唯一标识符，本地为文件路径哈希，在线为平台歌曲ID | `"local_abc123"` / `"net_456789"` |
| `title` | `String` | 歌曲标题 | `"晴天"` |
| `artist` | `String` | 歌手名称 | `"周杰伦"` |
| `album` | `String?` | 专辑名称（可空） | `"叶惠美"` |
| `duration` | `int` | 时长，毫秒 | `268000` |
| `coverUrl` | `String?` | 封面图URL（可空） | `"https://.../cover.jpg"` |
| `source` | `SongSource` | 来源枚举：`local` / `online` | `SongSource.local` |
| `filePath` | `String?` | 本地文件路径（仅本地歌曲） | `"/storage/emulated/0/Music/晴天.mp3"` |
| `onlineUrl` | `String?` | 在线播放URL（仅在线歌曲） | `"https://.../song.mp3"` |
| `lyrics` | `String?` | 歌词内容（lrc格式，可空） | `[00:00.00]歌词内容...` |
| `isFavorite` | `bool` | 是否收藏 | `false` |
| `playCount` | `int` | 播放次数 | `0` |
| `fileSize` | `int?` | 文件大小，字节（仅本地） | `4123456` |
| `mimeType` | `String?` | MIME类型（仅本地） | `"audio/mpeg"` |
| `createdAt` | `DateTime?` | 添加到库的时间 | `2024-01-15T10:30:00Z` |

### 枚举：SongSource

```dart
enum SongSource {
  local,   // 本地音乐
  online,  // 在线音乐
}
```

### Dart 实现

```dart
class SongModel {
  final String id;
  final String title;
  final String artist;
  final String? album;
  final int duration;          // milliseconds
  final String? coverUrl;
  final SongSource source;
  final String? filePath;     // for local songs
  final String? onlineUrl;    // for online songs
  final String? lyrics;
  final bool isFavorite;
  final int playCount;
  final int? fileSize;
  final String? mimeType;
  final DateTime? createdAt;

  // computed
  String get displayDuration => _formatDuration(duration);

  String _formatDuration(int ms) {
    final d = Duration(milliseconds: ms);
    return '${d.inMinutes}:${(d.inSeconds % 60).toString().padLeft(2, '0')}';
  }
}
```

## 2. PlaylistModel（歌单模型）

### 字段定义

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `id` | `String` | 歌单唯一标识符，UUID或哈希 | `"pl_abc123"` |
| `name` | `String` | 歌单名称 | `"我的心动歌单"` |
| `coverUrl` | `String?` | 封面图URL（可空） | `"https://.../cover.jpg"` |
| `songCount` | `int` | 歌曲数量 | `25` |
| `createTime` | `DateTime` | 创建时间 | `2024-01-15T10:30:00Z` |
| `updateTime` | `DateTime` | 最后修改时间 | `2024-02-20T14:00:00Z` |
| `description` | `String?` | 歌单描述（可空） | `"适合夜晚听的歌"` |
| `creator` | `String` | 创建者标识 | `"user_001"` |
| `type` | `PlaylistType` | 类型：`userCreated` / `onlineImported` | `PlaylistType.userCreated` |

### 枚举：PlaylistType

```dart
enum PlaylistType {
  userCreated,    // 用户自建
  onlineImported, // 在线导入（推荐歌单）
  favorite,       // 收藏歌单（系统内置）
  history,        // 播放历史（系统内置）
}
```

### Dart 实现

```dart
class PlaylistModel {
  final String id;
  final String name;
  final String? coverUrl;
  final int songCount;
  final DateTime createTime;
  final DateTime updateTime;
  final String? description;
  final String creator;
  final PlaylistType type;

  // relationship: songs (not stored in this model, queried via repository)
  List<SongModel>? songs;
}
```

## 3. LrcModel（歌词模型）

歌词按时间轴解析，支持多行同步。

### 字段定义

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `lines` | `List<LrcLine>` | 歌词行列表 |

### LrcLine 子结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `time` | `Duration` | 当前行开始时间 |
| `text` | `String` | 歌词文本（不含时间标签） |
| `endTime` | `Duration?` | 当前行结束时间（由下一行推导，可空） |

### Dart 实现

```dart
class LrcModel {
  final List<LrcLine> lines;

  /// 解析 LRC 文本字符串为 LrcModel
  factory LrcModel.parse(String lrcText) {
    final lines = <LrcLine>[];
    final regex = RegExp(r'\[(\d{2}):(\d{2})\.(\d{2,3})\](.*)');

    for (final rawLine in lrcText.split('\n')) {
      final match = regex.firstMatch(rawLine);
      if (match != null) {
        final min = int.parse(match.group(1)!);
        final sec = int.parse(match.group(2)!);
        final msStr = match.group(3)!;
        // 兼容 2 位（厘秒）和 3 位（毫秒）
        final ms = msStr.length == 2 ? int.parse(msStr) * 10 : int.parse(msStr);
        final time = Duration(minutes: min, seconds: sec, milliseconds: ms);
        final text = match.group(4)!.trim();
        lines.add(LrcLine(time: time, text: text));
      }
    }

    // 按时间排序
    lines.sort((a, b) => a.time.compareTo(b.time));

    // 补充 endTime
    for (int i = 0; i < lines.length - 1; i++) {
      lines[i] = lines[i].copyWith(endTime: lines[i + 1].time);
    }

    return LrcModel(lines: lines);
  }

  /// 根据播放进度获取当前行索引
  int getLineIndex(Duration position) {
    for (int i = lines.length - 1; i >= 0; i--) {
      if (position >= lines[i].time) return i;
    }
    return 0;
  }
}

class LrcLine {
  final Duration time;
  final String text;
  final Duration? endTime;

  LrcLine({required this.time, required this.text, this.endTime});
  LrcLine copyWith({Duration? endTime}) => LrcLine(
    time: time,
    text: text,
    endTime: endTime ?? this.endTime,
  );
}
```

## 4. SearchResultModel（搜索结果模型）

### 字段定义

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `keyword` | `String` | 搜索关键词 |
| `songs` | `List<SongModel>` | 歌曲结果列表 |
| `playlists` | `List<PlaylistModel>` | 歌单结果列表 |
| `artists` | `List<ArtistModel>` | 歌手结果列表（可选） |
| `totalSongCount` | `int` | 歌曲总数 |
| `totalPlaylistCount` | `int` | 歌单总数 |
| `page` | `int` | 当前页码 |
| `hasMore` | `bool` | 是否有更多 |

### ArtistModel（歌手模型，可选）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | `String` | 歌手ID |
| `name` | `String` | 歌手名 |
| `avatarUrl` | `String?` | 头像URL |
| `songCount` | `int` | 歌曲数量 |

### Dart 实现

```dart
class SearchResultModel {
  final String keyword;
  final List<SongModel> songs;
  final List<PlaylistModel> playlists;
  final List<ArtistModel>? artists;
  final int totalSongCount;
  final int totalPlaylistCount;
  final int page;
  final bool hasMore;

  bool get isEmpty => songs.isEmpty && playlists.isEmpty;
}
```

## 5. RankItemModel（排行榜条目）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `rank` | `int` | 排名 |
| `song` | `SongModel` | 歌曲信息 |

### 枚举：RankTrend

```dart
enum RankTrend {
  up,    // 上升
  down,  // 下降
  newEntry, // 新上榜
  same,  // 不变
}
```
