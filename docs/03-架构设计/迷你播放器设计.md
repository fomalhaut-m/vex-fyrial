# 底部迷你播放器（MiniPlayer）设计文档

> ⚠️ 本文档已更新为 Riverpod 版本（GetX 已移除）

## 1. 概述

MiniPlayer 是全局悬浮在页面底部的迷你播放器控件，始终可见且可交互，不受当前页面路由影响。

**设计目标**：
- 播放控制触手可及，切换页面不中断
- 轻量展示：封面 + 歌名 + 歌手 + 播放/暂停按钮
- 点击可切换到 Tab 0（播放器）

背景：白色 #FFFFFF
阴影：0, -2, 8, rgba(0,0,0,0.08)

### 2.2 状态

| 状态 | 表现 |
|------|------|
| 停止/无播放 | 隐藏，不占用屏幕空间 |
| 播放中 | 显示，播放按钮为暂停图标 |
| 暂停 | 显示，播放按钮为播放图标 |
| 加载中 | 播放按钮显示 CircularProgressIndicator |

## 3. 技术实现（Riverpod）

### 3.1 全局状态管理

MiniPlayer 是全局 Widget，通过 Riverpod 管理状态：

```dart
// providers/tab_index_provider.dart
final tabIndexProvider = NotifierProvider<TabIndexNotifier, TabIndexState>();

// providers/player_provider.dart
final playerProvider = NotifierProvider<PlayerNotifier, PlayerState>();
```

### 3.2 PlayerState 状态

```dart
class PlayerState {
  final List<SongModel> playlist;
  final int currentIndex;
  final SongModel? currentSong;
  final bool isPlaying;
  final bool isLoading;
  final int position;      // 毫秒
  final int duration;      // 毫秒
  final PlayMode playMode;
  final bool isShuffle;
  final String? error;

  double get progressPercent => duration > 0 ? position / duration : 0;
  String get currentTimeString => _formatDuration(position);
  String get totalTimeString => _formatDuration(duration);
}
```

### 3.3 MiniPlayer 显示/隐藏逻辑

```dart
// lib/widgets/mini_player.dart
class MiniPlayer extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final playerState = ref.watch(playerProvider);
    final tabState = ref.watch(tabIndexProvider);

    // 无播放内容时隐藏
    if (playerState.currentSong == null) {
      return const SizedBox.shrink();
    }

    // 在 Tab 0（播放器）时不显示 MiniPlayer
    if (tabState.currentIndex == 0) {
      return const SizedBox.shrink();
    }

    return // MiniPlayer 内容...
  }
}
```

### 3.4 MiniPlayer 点击行为

点击 MiniPlayer 切换到 Tab 0（播放器），不跳转到新页面：

```dart
onTap: () {
  // 点击切换到 Tab 0（播放器）
  ref.read(tabIndexProvider.notifier).switchTab(0);
}
```

## 4. 与其他组件的交互

| 组件 | 交互方式 |
|------|----------|
| `tabIndexProvider` | 管理当前 Tab 索引，MiniPlayer 据此决定显示/隐藏 |
| `playerProvider` | 管理播放状态、当前歌曲、进度等 |
| `HomePage` | 监听 `tabIndexProvider` 切换 Tab |

### MiniPlayer 在 HomePage 中的位置

```dart
// lib/app/modules/home/home_page.dart
bottomNavigationBar: Column(
  mainAxisSize: MainAxisSize.min,
  children: [
    // MiniPlayer：在 Tab 0（播放器）时不显示
    const MiniPlayer(),

    // 底部 Tab 导航栏
    BottomNavigationBar(...),
  ],
)
```

---

## 5. 交互行为

| 操作 | 行为 |
|------|------|
| 点击播放/暂停按钮 | `ref.read(playerProvider.notifier).togglePlayPause()` |
| 点击下一首按钮 | `ref.read(playerProvider.notifier).next()` |
| 点击 MiniPlayer 主体 | 切换到 Tab 0（播放器） |

## 6. 播放进度

MiniPlayer 进度条使用 `LinearProgressIndicator`：

```dart
LinearProgressIndicator(
  value: playerState.progressPercent.clamp(0.0, 1.0),
  minHeight: 2,
  backgroundColor: Colors.grey[200],
  color: AppTheme.primaryGreen,
)
```
