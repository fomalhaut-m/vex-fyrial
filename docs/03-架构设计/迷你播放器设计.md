# 底部迷你播放器（MiniPlayer）设计文档

## 1. 概述

MiniPlayer 是全局悬浮在页面底部的迷你播放器控件，始终可见且可交互，不受当前页面路由影响。

**设计目标**：
- 播放控制触手可及，切换页面不中断
- 轻量展示：封面 + 歌名 + 歌手 + 播放/暂停按钮
- 点击可展开为完整的播放详情页

|------|------|

## 3. 技术实现

### 3.1 全局状态管理

MiniPlayer 不是一个独立页面，而是一个 **全局 Widget**，挂载在 App 根节点：

```dart
// main.dart
GetMaterialApp(
  // ...
  builder: (context, child) {
    return Stack(
      children: [
        child!,          // 当前页面内容
        Positioned(      // MiniPlayer 全局浮层
          left: 0,
          right: 0,
          bottom: 0,
          child: MiniPlayer(),
        ),
      ],
    );
  },
)
```

MiniPlayer 的显示/隐藏、播放状态、当前歌曲均由 `PlayerController`（GetX）统一管理，无需跨模块通信。

### 3.2 PlayerController 状态

```dart
class PlayerController extends GetxController {
  // 当前播放歌曲
  Rx<Song?> currentSong = Rx<Song?>(null);

  // 播放状态
  Rx<PlayerState> playerState = PlayerState.stopped.obs;

  // 播放进度（0.0 - 1.0）
  RxDouble progress = 0.0.obs;

  // 当前播放位置（秒）
  RxInt currentPosition = 0.obs;

  // 迷你播放器是否可见（当前有歌曲且不在播放页）
  bool get isMiniPlayerVisible =>
      currentSong.value != null && playerState.value != PlayerState.stopped;
}
```

### 3.3 与 PlayerController 的交互

MiniPlayer 直接 `Get.find<PlayerController>()` 绑定响应式状态：

```dart
class MiniPlayer extends GetView<PlayerController> {
  @override
  Widget build(BuildContext context) {
    return Obx(() {
      if (!controller.isMiniPlayerVisible) {
        return SizedBox.shrink();
      }
      return _MiniPlayerContent(
        song: controller.currentSong.value!,
        isPlaying: controller.playerState.value == PlayerState.playing,
        progress: controller.progress.value,
        onPlayPause: controller.togglePlay,
        onSkipNext: controller.playNext,
        onSkipPrev: controller.playPrevious,
        onTap: () => Get.toNamed(AppRoutes.PLAYER_PAGE),
      );
    });
  }
}
```

### 3.4 页面层级冲突处理

当用户进入 **Player 播放页**（/pages/player-page）时，MiniPlayer 应自动隐藏，避免重复显示：

```dart
// 在 PlayerController 中判断
bool get isMiniPlayerVisible {
  if (currentSong.value == null) return false;
  // 当前路由为播放页时隐藏 MiniPlayer
  if (Get.currentRoute == AppRoutes.PLAYER_PAGE) return false;
  return true;
}
```

## 4. 交互行为

| 操作 | 行为 |
|------|------|
| 点击播放/暂停按钮 | 调用 `playerController.togglePlay()` |
| 点击下一首/上一首 | 调用 `playerController.playNext()` / `playPrevious()` |
| 点击 MiniPlayer 主体区域 | 跳转到 `/pages/player-page`（完整播放页） |
| 滑動进度条 | 拖动 seek 到对应位置 |
| 播放结束 | 自动播放下一首，若队列为空则停止并隐藏 |

## 5. 动画

| 场景 | 动画 |
|------|------|
| 出现/消失 | 垂直滑动 + 淡入淡出，duration 250ms |
| 播放状态切换 | 播放/暂停图标使用 AnimatedSwitcher |
| 进度条 | 连续线性更新，不卡顿 |
| 封面图片 | CrossFade 切换，无白边 |
