# Stats Page - 播放统计页面

## 页面概述

播放统计页面承载 PRD 中的「播放统计」功能，支持两个统计维度：
1. **按歌曲统计**：每首歌的播放次数、总时长、主播放时段、最爱片段
2. **按分类统计**：各类别音乐的累计播放次数、总时长、时段分布

### 组件层级

```
StatsBySongPage
├── SummaryCard (顶部汇总)
├── SearchBar (搜索框)
├── SortSelector (排序选项)
└── ListView (统计列表)
    └── SongStatsTile (歌曲统计项)
        ├── CoverImage
        ├── SongInfo (歌名/歌手)
        ├── PlayCount (次数 + 可视化条)
        ├── TotalDuration
        └── FavoriteInfo (时段 + 片段)
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击歌曲统计项 | 跳转歌曲详情页（展示该歌曲详细统计） |
| 点击播放次数可视化条 | 无操作（装饰性） |
| 点击排序选项 | 切换排序方式，列表刷新 |
| 点击搜索 | 过滤显示匹配的歌曲 |
| 下拉刷新 | 刷新统计数据 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 加载中 | 骨架屏 |
| 空状态（无统计） | "还没有播放记录，开始听歌吧" |
| 搜索无结果 | "未找到匹配的歌曲" |

```
│  时段分布                           │  ⚠️ 按 morning/afternoon/evening/night 展示
│  🌙 深夜  312次 (25%)               │
│  🌆 傍晚  456次 (37%)               │

│  🌅 上午  168次 (14%)               │
├─────────────────────────────────────┤
│  该分类下的歌曲                     │
│  [封面] 歌曲A    播放 128次  ▶     │
│  [封面] 歌曲B    播放  95次  ▶     │
│  ...                                │
└─────────────────────────────────────┘
```

### 组件层级

```
StatsByGenrePage
├── GenreTabBar (横向分类 Tab)
├── GenreSummaryCard (分类汇总)
│   ├── GenreName
│   ├── TotalPlayCount (次数 + 可视化条)
│   ├── TotalDuration
│   └── SongCount
├── TimeSlotChart (时段分布图表)   ⚠️ PRD v3 Tab 3 定义：按时段
│   └── TimeSlotBar (时段柱状图)
└── SongList (该分类下歌曲列表)
    └── SongStatsTile
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击分类 Tab | 切换分类，显示该分类统计 |
| 点击歌曲列表项 | 播放该歌曲 |
| 横滑分类 Tab | 切换分类 |
| 下拉刷新 | 刷新统计数据 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 加载中 | 骨架屏 |
| 无分类 | "还未给歌曲分类，在歌曲详情页添加分类" |
| 某分类无歌曲 | "该分类下暂无歌曲" |

## 歌曲详情弹窗（统计）

### 页面结构

```
┌─────────────────────────────────────┐
│                              [✕]   │  关闭按钮
├─────────────────────────────────────┤
│  [封面大图]                         │
│  歌曲名称                           │
│  歌手 · 专辑                        │
├─────────────────────────────────────┤
│  播放 128 次  ·  累计 6.4 小时      │
├─────────────────────────────────────┤
│  主要播放时段                        │
│  🌙 深夜 (22:00 - 06:00)           │
├─────────────────────────────────────┤
│  最爱片段                           │
│  2:00 - 2:30  ·  被重复播放 8 次    │
│  [▶ 从这里播放]                     │
├─────────────────────────────────────┤
│  歌曲分类                           │
│  [流行] [电子]    [编辑]            │
├─────────────────────────────────────┤
│  [从这首歌开始播放全部]              │  主按钮
└─────────────────────────────────────┘
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击"从这里播放" | seek 到最爱片段开始播放 |
| 点击"从这首歌开始播放全部" | 播放整张列表，以该歌曲为起点 |
| 点击编辑分类 | 弹出分类编辑框 |
| 点击关闭 | 关闭弹窗 |

---

## 通用交互规范

| 规范 | 说明 |
|------|------|
| 播放次数可视化条 | 最长条 = 该分类/全局最多播放次数，百分比展示 |
| 时段分布图表 | 横向柱状图，每段 25% 等分 |
| 最爱片段展示 | 显示"分:秒 - 分:秒"，点击直接 seek 播放 |
| 分类颜色 | 每个分类对应一个预设颜色，列表中图标颜色跟随分类色 |

---

## 时段定义

| 时段 | 时间范围 | 图标 |
|------|---------|------|
| 上午 (morning) | 06:00 - 12:00 | 🌅 |
| 下午 (afternoon) | 12:00 - 18:00 | ☀️ |
| 傍晚 (evening) | 18:00 - 24:00 | 🌆 |
| 深夜 (night) | 00:00 - 06:00 | 🌙 |

---

## 性能优化

- 统计数据异步批量读取，不阻塞 UI
- 列表使用 ListView.builder 懒加载
- 分类 Tab 使用 RepaintBoundary 隔离重绘
