# Search Page - 搜索页

## 页面概述

搜索页支持搜索本地音乐库中的歌曲。提供搜索历史、搜索建议、搜索结果展示等功能。

## 搜索输入框

### 规格

| 属性 | 值 |
|------|---|
| 高度 | 40dp |
| 背景色 | #1E1E1E |
| 圆角 | 20dp |
| 左侧图标 | 🔍，#727272 |
| 右侧清除按钮 | ✕，#727272，输入时有文字才显示 |
| placeholder | "搜索歌曲、歌手、专辑..."，#727272 |

### 交互

| 交互 | 行为 |
|------|------|
| 点击输入框 | 聚焦键盘弹出，显示光标 |
| 输入文字 | 实时触发搜索建议（300ms 防抖） |
| 点击清除按钮 | 清空输入框，恢复默认状态 |
| 点击返回按钮 | 关闭搜索页，返回上一页 |

---

## 搜索历史

### 存储规则

- 本地存储，最多 50 条
- 超出后删除最早的
- 搜索时去重（不重复添加）

### 交互

| 交互 | 行为 |
|------|------|
| 点击历史项 | 执行搜索 |
| 点击清空按钮 | 确认弹窗后清空全部 |
| 左滑历史项 | 删除单条 |

---

## 搜索建议

### 建议类型

| 类型 | 图标 | 示例 |
|------|------|------|
| 本地歌曲 | 📂 | "夜曲.mp3 - 周杰伦" |
| 本地专辑文件夹 | 📁 | "本地专辑: 周杰伦合集" |

### 防抖规则

- 输入后等待 300ms 无新输入才触发搜索
- 快速输入时取消前一个请求

| 状态 | 显示 |

## 技术规范

### 搜索防抖

```dart
// 300ms 防抖
Timer? _debounceTimer;
void onSearchChanged(String query) {
  _debounceTimer?.cancel();
  _debounceTimer = Timer(Duration(milliseconds: 300), () {
    _performSearch(query);
  });
}
```

### 本地搜索（纯本地）

```dart
// 本地音乐搜索，扫描设备存储中的音乐文件
List<Song> localSearch(String query) {
  return localSongs.where((song) {
    return song.name.contains(query) ||
           song.artist.contains(query) ||
           song.album.contains(query) ||
           song.filePath.contains(query);
  }).toList();
}
```
