# Home Tabs - 首页 Tab 布局规范

> ⚠️ 本文档已按 PRD v3 修订：App 底部导航重构为 **4 Tab 底部导航**
> - Tab 1：播放器（Player）
> - Tab 2：播放列表（Playlist）
> - Tab 3：统计（Stats）
> - Tab 4：设置（Settings）

## 变更说明

| 版本 | Tab 结构 |
|------|----------|
| PRD v2 | 本地音乐 \| 歌单 \| OSS同步 \| 设置 |
| PRD v3 | 播放器 \| 播放列表 \| 统计 \| 设置 |

**核心变化：**
- 新增「播放器」Tab（Tab 1），承载当前播放内容（封面/歌词/播控）
- 原「本地音乐 + 我的歌单」合并为「播放列表」Tab（Tab 2）
- 新增「统计」Tab（Tab 3），承载播放统计功能
- 原「OSS同步」从独立 Tab 降级为「设置」Tab 内的配置项

---

## Tab 1：播放器（Player）

### 页面结构

```
┌─────────────────────────────────────┐
│  当前播放内容                         │  页面标题（可选，不显示）
├─────────────────────────────────────┤
│                                     │
│      ┌─────────────────────┐        │
│      │                     │        │
│      │     Album Cover     │        │  封面大图，正方形，圆角 8dp
│      │    (280dp × 280dp)  │        │
│      │                     │        │
│      └─────────────────────┘        │
│                                     │
│          歌曲名称                    │  song-title 样式，居中
│          歌手名                      │  artist-name 样式，居中，#B3B3B3
│                                     │
├─────────────────────────────────────┤
│  0:00 ▓▓▓▓▓▓▓░░░░░░░░░░░░ 3:45   │  进度条 + 时间
├─────────────────────────────────────┤
│     ⏮      ▶/⏸       ⏭             │  控制栏
│           🔀        🔁               │  随机 / 循环模式
└─────────────────────────────────────┘
```

### 组件层级

```
PlayerTab
├── CoverView (封面大图 + 旋转动画)
│   └── AlbumCover (300dp × 300dp，圆角 8dp)
├── SongInfo (歌曲名称 + 歌手名)
├── ProgressBar (进度条)
│   ├── SliderTrack
│   ├── SliderThumb
│   ├── CurrentTime
│   └── TotalTime
└── ControlBar (控制栏)
    ├── ShuffleButton (随机播放)
    ├── PreviousButton (上一首)
    ├── PlayPauseButton (播放/暂停)
    ├── NextButton (下一首)
    └── RepeatButton (循环模式)
```

### 歌词模式（可切换）

```
┌─────────────────────────────────────┐
│  歌曲名称                            │
│  歌手名                              │
├─────────────────────────────────────┤
│  这是第一句歌词（上一行，#727272）    │
│  这是第二句歌词（当前行，#FFFFFF）    │  当前行字号加大，高亮
│  这是第三句歌词（下一行，#727272）    │
│  这是第四句歌词                      │
├─────────────────────────────────────┤
│  0:00 ▓▓▓▓▓▓▓░░░░░░░░░░░░ 3:45   │
├─────────────────────────────────────┤
│     ⏮      ▶/⏸       ⏭             │
│           🔀        🔁               │
└─────────────────────────────────────┘
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击封面 | 切换到歌词模式（300ms 渐变） |
| 点击歌词区域 | 切换到封面模式 |
| 左右滑封面 | 切歌（上一首/下一首） |
| 点击播放/暂停 | 切换播放状态 |
| 点击上一首/下一首 | 切歌 |
| 点击随机/循环 | 切换播放模式 |
| 下拉手势 | 无操作（Tab 内无全屏下滑关闭） |

### 状态说明

| 状态 | 显示 |
|------|------|
| 无播放内容 | 显示默认封面 + "未选择歌曲" |
| 正在播放 | 播放按钮显示暂停图标，封面旋转 |
| 已暂停 | 播放按钮显示播放图标，封面停止旋转 |
| 加载中 | 封面显示加载动画 |

## Tab 2：播放列表（Playlist）

### 页面结构

```
┌─────────────────────────────────────┐
│  播放列表               [🔍 搜索]    │  页面标题 + 搜索按钮
├─────────────────────────────────────┤
│  [全部] [最近播放] [收藏]            │  子 Tab 筛选
├─────────────────────────────────────┤
│  共 1,234 首  ·  总时长 128:35       │  统计信息
├─────────────────────────────────────┤
│  [音乐图标] 歌曲名 A    歌手A  3:45  │  歌曲列表
│  [音乐图标] 歌曲名 B    歌手B  4:12  │
│  [音乐图标] 歌曲名 C    歌手C  2:58  │
│  ...                                │
└─────────────────────────────────────┘
```

### 组件层级

```
PlaylistTab
├── AppBar (标题 + 搜索按钮)
├── SubTabBar (全部/最近播放/收藏)
├── StatsBar (歌曲数量 + 总时长)
├── SongListView (歌曲列表)
│   └── SongTile × N (图标+歌名+歌手+时长)
└── PlaylistEntranceCard (歌单入口卡片)
```

### 歌单入口卡片

```
┌─────────────────────────────────────┐
│  我的歌单                            │
│  ┌─────────┐ ┌─────────┐            │
│  │ 歌单封面 │ │ 歌单封面 │            │
│  │ 歌单名   │ │ 歌单名   │            │
│  │ 12首     │ │ 8首      │            │
│  └─────────┘ └─────────┘            │
└─────────────────────────────────────┘
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击歌曲 | 播放该歌曲 |
| 长按歌曲 | 显示操作菜单（收藏/添加到歌单/删除/详情） |
| 点击子 Tab | 切换显示全部/最近播放/收藏 |
| 点击搜索 | 展开搜索框 |
| 点击歌单卡片 | 跳转歌单详情页 |
| 点击右上角扫描 | 触发本地音乐扫描 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 未扫描 | "点击右上角扫描本地音乐" |
| 扫描中 | 进度条 "已扫描 123/1000 首" |
| 扫描完成 | 显示歌曲列表 + 统计信息 |
| 空状态 | "未发现本地音乐，请检查存储权限" |

```

├── GenreTabBar (横向分类 Tab)

├── TimeSlotChart (时段分布图表)
└── SongList (该分类下歌曲列表)
    └── SongStatsTile
```

### 交互说明

| 交互 | 行为 |
|------|------|
| 点击时间维度 Tab | 切换按日/周/月统计 |
| 点击分类 Tab | 切换分类，显示该分类统计 |
| 点击歌曲列表项 | 播放该歌曲 |
| 横滑分类 Tab | 切换分类 |
| 下拉刷新 | 刷新统计数据 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 加载中 | 骨架屏 |
| 无数据 | "还没有播放记录，开始听歌吧" |
| 无分类 | "还未给歌曲分类" |

## 底部迷你播放器（MiniPlayer）

> ⚠️ 全局组件，所有 Tab 间常驻显示

### 尺寸规范

| 属性 | 值 |
|------|---|
| 高度 | 64dp |
| 封面尺寸 | 48dp × 48dp |
| 封面圆角 | 4dp |
| 左侧边距 | 12dp |

### 结构

```
┌──────────────────────────────────────────────────────────┐
│ [封面] 歌曲名称                      [播放/暂停] [下一首] │
│         歌手名 · 02:35                    icon  icon     │
└──────────────────────────────────────────────────────────┘
```

### 交互规范

| 交互 | 行为 |
|------|------|
| 点击 MiniPlayer 任意区域 | 打开全屏播放页（PlayerPage） |
| 点击播放/暂停按钮 | 切换播放状态 |
| 点击下一首按钮 | 切到下一首 |

### 进度指示

- MiniPlayer 顶部显示 2dp 高度进度条
- 颜色：#1DB954
- 背景：rgba(255,255,255,0.1)

## 通用交互规范

| 规范 | 说明 |
|------|------|
| 点击热区 | 最小 48dp × 48dp |
| 按压反馈 | 可点击元素按压时透明度变为 0.7 |
| 跳转动画 | 从右滑入，300ms ease-in-out |
| Tab 切换 | 即时切换，无动画 |
| MiniPlayer | 播放时自动显示，无歌曲时隐藏 |
