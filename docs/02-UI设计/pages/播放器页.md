# Player Page - 全屏播放页

## 页面概述

全屏播放页是 App 的核心页面，展示当前播放歌曲的封面、歌词（可选）、控制栏。是用户交互最频繁的页面之一。

## 交互说明

### 顶部操作栏

| 交互 | 行为 |
|------|------|
| 点击返回按钮 | 关闭播放页，返回上一页面 |
| 点击更多按钮 | 显示下拉菜单（播放列表、设为铃声等） |
| 点击喜欢按钮 | 切换喜欢状态，红心动画 |

### 封面/歌词区域

| 交互 | 行为 |
|------|------|
| 点击封面 | 切换到歌词模式（300ms 渐变） |
| 点击歌词区域 | 切换到封面模式 |
| 双击封面 | 收藏/取消收藏 |
| 在歌词区域上下滑动 | 手动滚动歌词，自动滚动取消 |
| 点击歌词行 | 跳转到该歌词时刻播放 |

### 进度条

| 交互 | 行为 |
|------|------|
| 点击进度条任意位置 | 跳转到对应时间 |
| 拖动滑块 | 实时预览，松手后跳转 |
| 显示进度 | 播放时实时更新 |

### 控制栏

| 交互 | 行为 |
|------|------|
| 点击播放/暂停 | 切换播放状态，图标动画切换 |
| 点击上一首 | 播放上一首 |
| 点击下一首 | 播放下一首 |
| 点击随机播放 | 开关随机播放，图标高亮/取消 |
| 点击循环模式 | 切换：全部循环 → 单曲循环 → 关闭循环 |

### 额外操作

| 交互 | 行为 |
|------|------|
| 点击分享 | 显示分享面板 |
| 点击歌词按钮 | 切换封面/歌词模式 |

---

## 手势交互

| 手势 | 区域 | 行为 |
|------|------|------|
| 向下滑动 | 任意区域 | 关闭播放页（返回上一页面） |
| 左右滑动 | 封面区域 | 切歌（上一首/下一首） |
| 双指缩放 | 封面区域 | 放大/缩小封面 |
| 长按封面 | 封面区域 | 显示快捷操作菜单 |

## 状态说明

### 播放状态

| 状态 | 显示 |
|------|------|
| 正在播放 | 播放按钮显示暂停图标，进度条前进 |
| 已暂停 | 播放按钮显示播放图标，进度条停止 |
| 加载中 | 播放按钮显示加载动画（转圈） |
| 播放错误 | 显示 toast "播放失败，请重试" |

### 歌词状态

| 状态 | 显示 |
|------|------|
| 有歌词 | 显示歌词，按当前时间高亮 |
| 无歌词 | 显示"暂无歌词"提示 |
| 歌词加载中 | 显示加载动画 |
| 歌词加载失败 | 显示"歌词加载失败" |

### 喜欢状态

| 状态 | 显示 |
|------|------|
| 已喜欢 | 心形图标红色填充 ❤️ |
| 未喜欢 | 心形图标白色描边 ♡ |

### 循环模式状态

| 状态 | 图标 | 说明 |
|------|------|------|
| 全部循环 | 🔁 | 播完最后一首回到第一首 |
| 单曲循环 | 🔂 | 循环播放当前歌曲 |
| 关闭循环 | 🔁 + 划线 | 播完停止 |

## 动画规范

| 动作 | 时长 | 缓动函数 |
|------|------|----------|
| 播放/暂停图标切换 | 200ms | ease-in-out |
| 封面切换（切歌） | 400ms | ease-in-out，scale + fade |
| 封面/歌词模式切换 | 300ms | ease-in-out，fade |
| 歌词滚动 | 与播放同步 | linear |
| 歌词高亮切换 | 150ms | ease-out |
| 心形收藏动画 | 300ms | scale 1 → 1.3 → 1，弹跳 |
| 页面打开（从底部） | 300ms | ease-in-out |
| 页面关闭（滑下） | 300ms | ease-in-out |

### 封面旋转动画（可选）

| 状态 | 动画 |
|------|------|
| 播放中 | 封面缓慢旋转，360°/30s |
| 暂停中 | 旋转停止，保持当前角度 |
| 切歌 | 旋转重置到 0°，重新开始 |

## 细节规范

### 封面图片

- 尺寸：宽度 = 屏幕宽度 - 64dp，高度 = 宽度（正方形）
- 圆角：8dp
- 阴影：`BoxShadow(color: rgba(0,0,0,0.5), blurRadius: 30, offset: Offset(0, 15))`
- 占位图：灰色渐变 + 音符图标

### 进度条

- 轨道高度：4dp
- 滑块直径：12dp
- 滑块颜色：#FFFFFF
- 已播放颜色：#1DB954
- 未播放轨道颜色：rgba(255,255,255,0.3)

### 控制按钮尺寸

| 按钮 | 尺寸 |
|------|------|
| 播放/暂停 | 64dp × 64dp |
| 上一首/下一首 | 48dp × 48dp |
| 随机/循环/分享/歌词 | 40dp × 40dp |

---

## 空状态与错误状态

| 状态 | 显示 |
|------|------|
| 无播放内容 | 显示"未选择歌曲"，返回按钮 |
| 歌曲加载中 | 封面显示加载动画，控件禁用 |
| 播放失败 | Toast 提示"播放失败"，显示重试按钮 |
| 封面加载失败 | 显示灰色占位图 + 音符图标 |
