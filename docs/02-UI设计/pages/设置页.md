# Settings Page - 设置页

## 页面概述

设置页是 App 的 **Tab 4（设置）**，提供播放设置、OSS 同步配置、存储管理等各项功能。

> ⚠️ v3 修订：OSS 同步配置从独立 Tab 降级为设置页内的配置项

## 设置项分类

### 1. 播放设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 随机播放 | Switch | 关 | 新歌曲列表播放顺序 |
| 列表循环 | Switch | 开 | 播完列表后从头开始 |
| 断点续播 | Switch | 开 | 重新打开 App 从上次位置继续 |
| 播放音质 | Selector | 高品质 | 自动/低音质/标准/高品质/无损 |
| 倍速播放 | Selector | 1.0x | 0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x |

### 2. OSS 同步配置 ⚠️ v3 新增（从独立 Tab 降级）

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| Bucket | TextInput | 空 | OSS Bucket 名称 |
| Endpoint | TextInput | 空 | OSS Endpoint 地址 |
| AK/SK | SecureInput | 空 | Access Key / Secret Key |
| 同步目录 | Navigation | 空 | 选择本地同步目录 |
| 同步频率 | Selector | 每30分钟 | 每15分钟 / 每30分钟 / 每小时 / 手动 |
| 自动同步 | Switch | 关 | 开启/关闭自动同步 |

### 3. 通知与提醒

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 新歌提醒 | Switch | 开 | 本地音乐库新增歌曲时通知 |

### 3. 存储与数据

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 缓存管理 | 显示已用 + 清理按钮 | 点击显示缓存详情，可一键清理 |
| 下载位置 | 显示当前路径 | 点击切换存储位置 |
| WiFi 下自动下载 | Switch | 仅 WiFi 环境自动下载 |

### 4. 通用设置

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 深色模式 | Selector | 跟随系统 / 始终开启 / 始终关闭 |
| 歌词字体大小 | Selector | 小 / 中 / 大 |
| 语言 | Selector | 简体中文 / 繁体中文 / English |
| 关于 | 跳转 | 显示 App 版本、版权信息 |
| 隐私政策 | 跳转 | 打开隐私政策页面 |
| 用户协议 | 跳转 | 打开用户协议页面 |

## 交互说明

### Switch 开关

```
┌─────────────────────────────────────┐
│ ◉ 设置项名称             [Switch]   │
└─────────────────────────────────────┘
```

| 交互 | 行为 |
|------|------|
| 点击 Switch | 切换开关状态，保存设置 |
| 点击设置项名称 | 无操作（仅 Switch 可点击） |

### Selector 选择器

```
┌─────────────────────────────────────┐
│ ◉ 设置项名称             [当前值 ▼] │
└─────────────────────────────────────┘
```

| 交互 | 行为 |
|------|------|
| 点击设置项 / 箭头 | 展开下拉选项 |
| 选择选项 | 关闭下拉，保存设置 |

### 下拉选项展开

```
┌─────────────────────────────────────┐
│ ◉ 深色模式             [跟随系统 ▼] │
│   ▼ 跟随系统                       │
│     始终开启                        │
│     始终关闭                        │
└─────────────────────────────────────┘
```

### 跳转项

```
┌─────────────────────────────────────┐
│ ◉ 关于                       >       │
└─────────────────────────────────────┘
```

| 交互 | 行为 |
|------|------|
| 点击设置项 | 跳转到对应页面（从右滑入） |

## 列表项组件类型

### Switch 型

```
高度 56dp
左侧：[图标可选] 文字
右侧：Switch
```

### Selector 型

```
高度 56dp
左侧：[图标可选] 文字
右侧：当前值 + ▼ 箭头
点击区域：整行
```

### Navigation 型

```
高度 56dp
左侧：[图标可选] 文字
右侧：值（如有）+ > 箭头
点击区域：整行
```
