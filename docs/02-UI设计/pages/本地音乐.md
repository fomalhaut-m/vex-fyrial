# Local Music Page - 本地音乐页

## 页面概述

本地音乐页用于管理和播放设备本地的音频文件。提供扫描、分类、搜索、批量管理等功能。

---

```
│ [<]  本地音乐              [🔍] [⋮] │  导航栏 + 搜索 + 更多
├─────────────────────────────────────┤
```

## 筛选与排序

### 筛选栏

```
┌─────────────────────────────────────┐
│ 全部 | 歌手 | 专辑 | 文件夹          │  横向滚动 Tab
└─────────────────────────────────────┘
```

| Tab | 说明 |
|-----|------|
| 全部 | 显示所有本地音乐 |
| 歌手 | 按歌手分组显示 |
| 专辑 | 按专辑分组显示 |
| 文件夹 | 按存储路径分组显示 |

### 排序选项（点击右上角 ⋮）

```
排序方式
○ 文件名
○ 歌曲名
○ 歌手
○ 时长
○ 添加时间
○ 最近播放
```

## 歌曲列表项

### 常规列表项

```
┌─────────────────────────────────────────────────────┐
│ ♪ │ 歌曲名称        [封面]  歌手名      3:45  [⋮]   │  ← 标准布局
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│ ▶ │ 歌曲名称 (正在播放)  歌手名      3:45  [⋮]     │  ← 当前播放
│   │ ▓▓▓▓░░░░░░░░░░░░░░░░░░               │  ← 进度条（当前播放项下方）
└─────────────────────────────────────────────────────┘
```

### 列表项元素

| 元素 | 宽度 | 样式 |
|------|------|------|
| 播放指示器 | 16dp | 播放中显示 ▶，选中显示 ♪ |
| 歌曲名 | 自适应 | body1，#FFFFFF，最大两行省略 |
| 封面缩略图 | 40dp × 40dp | 圆角 4dp，缺失显示音符占位图 |
| 歌手名 | 80dp | body2，#B3B3B3，单行省略 |
| 时长 | 40dp | caption，#727272 |
| 更多按钮 | 40dp | ⋮ 图标，展开操作菜单 |

### 列表项状态

| 状态 | 显示 |
|------|------|
| 默认 | 白底黑字 |
| 当前播放 | 左侧绿色指示器，歌曲名高亮 |
| 长按选中 | 左侧复选框，打断播放 |
| 下载中 | 时长显示下载进度 |
| 播放失败 | 红色感叹号图标 |

## 交互说明

### 基本操作

| 交互 | 行为 |
|------|------|
| 点击列表项 | 播放该歌曲，全屏打开播放页 |
| 点击 ⋮ 按钮 | 显示操作菜单（播放、添加歌单、设为铃声、删除） |
| 长按列表项 | 进入多选模式，显示复选框 |
| 左滑列表项 | 快速操作：添加到歌单 / 删除 |
| 右滑列表项 | 快速播放（不打断当前播放） |

### 操作菜单

```
┌─────────────────────────────────────┐
│ ▶ 播放                             │
│ ▶ 下一首播放                       │
│ ➕ 添加到歌单                      │
│ 🔔 设为铃声                       │
│ 📝 编辑信息                       │
│ 🗑️ 删除                           │
└─────────────────────────────────────┘
```

### 多选模式

| 交互 | 行为 |
|------|------|
| 点击列表项（多选模式） | 切换选中状态 |
| 点击全选 | 全选/取消全选 |
| 点击播放 | 播放选中项 |
| 点击添加 | 批量添加到歌单 |
| 点击取消 | 退出多选模式 |
| 点击删除 | 确认弹窗，删除选中项 |

### 排序与筛选

| 交互 | 行为 |
|------|------|
| 点击筛选 Tab | 切换筛选方式，列表刷新 |
| 点击排序选项 | 切换排序方式，列表重排 |

## 扫描功能

### 扫描入口


### 扫描状态

| 状态 | 显示 |
|------|------|
| 未扫描 | 显示扫描入口卡片 |
| 扫描中 | 显示进度 "正在扫描... 已找到 123 首" |
| 扫描完成 | 显示统计 "共扫描到 1,234 首音乐" |

## 状态说明

| 状态 | 显示 |
|------|------|
| 加载中 | 骨架屏 |

| 加载失败 | 错误页 + 重试按钮 |
