# Splash & Guide Pages - 闪屏页与引导页

## 页面概述

| 页面 | 触发场景 | 时长/行为 |
|------|----------|----------|
| Splash（闪屏页） | App 冷启动 | 1.5s 或加载完成 |
| Guide（引导页） | 首次安装 App | 滑动切换，3-4 页 |

---

## 1. 闪屏页 (Splash Page)

### 页面结构

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│           [品牌 Logo]               │  居中，白色 Spotify 风格音符图标
│           Vexfy              │  品牌名称，h2 样式，白色
│                                     │
│                                     │
│         正在加载...                 │  底部，caption 样式，#727272
│         ████████░░░░               │  加载进度条，#1DB954
└─────────────────────────────────────┘
```

### 设计规范

| 元素 | 规范 |
|------|------|
| 背景色 | #121212（深黑） |
| Logo | 音符图标 + "Vexfy" 文字，垂直居中偏上 |
| 加载提示 | 底部 80dp 处，caption 样式 |
| 加载进度条 | 高度 2dp，宽度随加载进度增长，#1DB954 |

### 动画规范

| 动作 | 规范 |
|------|------|
| Logo 出现 | 300ms fade-in，从 alpha 0 → 1 |
| 加载进度 | 循环动画，从 0% → 100%，每 1.5s 一周期 |
| 页面切换 | 300ms fade-out，自动跳转到主页或引导页 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 加载中 | 显示进度条动画 |
| 加载失败 | 显示重试按钮 "重新加载"，点击重新初始化 |
| 无网络 | 显示 "网络不可用，请检查网络设置" |

## 2. 引导页 (Guide Pages)

### 页面结构

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│         [插画/图标]                 │  居中，120dp × 120dp 圆形容器
│                                     │
│         标题文字                    │  h2 样式，居中，#FFFFFF
│                                     │
│         描述文字                    │  body1 样式，居中，#B3B3B3
│                                     │
│                                     │
│     ○ ○ ○ ○                        │  页码指示器，圆点
│                                     │
│         [跳过]      [下一步]        │  底部操作按钮
└─────────────────────────────────────┘
```

### 引导页内容（4页）

#### 第1页：欢迎使用

```
图标：[音乐符号 + 手机图标组合]
标题：沉浸式音乐体验
描述：一触即达，畅享本地与在线海量音乐
```

#### 第2页：极简播放

```
图标：[耳机 + 波形图组合]
标题：极简播放界面
描述：专注于音乐，告别繁杂功能
```

#### 第3页：本地+云端，音乐无忧

```
图标：[云存储 + 音符图标组合]
标题：本地 + 云端
描述：私有OSS存储，音乐永不丢失，本地与云端双向同步
```

#### 第4页：开始体验

```
图标：[音符 + 闪电组合]
标题：准备好探索了吗？
描述：点击下方按钮，开启你的音乐之旅
按钮：[开始体验] 全宽按钮，#1DB954 背景
```

### 交互规范

| 交互 | 行为 |
|------|------|
| 左右滑动 | 切换引导页，带 300ms 滑动动画 |
| 点击"下一步" | 切换到下一页，最后一页跳转到首页 |
| 点击"跳过" | 直接跳转到首页 |
| 页码指示器 | 当前页高亮（#1DB954），其他页（#727272） |
| 滑动到最后页 | "下一步" 变为 "开始体验" |

### 动画规范

| 动作 | 规范 |
|------|------|
| 页面切换 | 300ms ease-in-out，水平滑动 |
| 图标出现 | 400ms fade-in + scale，从 0.8 → 1.0 |
| 文字出现 | 300ms fade-in，延迟 200ms |
| 指示器切换 | 200ms 颜色变化 |
| 按钮点击 | 100ms scale，从 1.0 → 0.95 → 1.0 |

### 状态说明

| 状态 | 显示 |
|------|------|
| 首页 | 不显示跳过按钮，显示"开始体验"按钮 |
| 非首页 | 显示跳过按钮 + 下一步按钮 |

## 3. 首次启动判断逻辑

```dart
// 判断是否首次启动
bool isFirstLaunch = await getIsFirstLaunch(); // 读取本地存储

if (isFirstLaunch) {
  // 跳转引导页
  Navigator.pushReplacementNamed(context, '/guide');
} else {
  // 跳转首页
  Navigator.pushReplacementNamed(context, '/home');
}
```

### 本地存储

| Key | 类型 | 说明 |
|-----|------|------|
| `is_first_launch` | bool | 是否首次启动，false 表示已看过引导页 |
| `guide_completed_at` | DateTime | 引导页完成时间（可选，用于分析） |

---

## 4. 设计与开发注意事项

1. **闪屏页不可跳过**：用户看到完整品牌展示
2. **引导页支持手势滑动**：原生 iOS/Android 交互体验
3. **数据预加载**：引导页展示期间，后台预加载首屏数据
4. **内存管理**：引导页完成后释放图片资源
5. **熄屏处理**：引导页期间禁止熄屏，超时保持白屏
