# Color Theme - 配色规范

> ⚠️ v2 修订：统一所有文档的颜色变量命名，建立明确命名空间映射。

## 命名规范（统一标准）

| 命名空间 | 用途 | 示例 |
|---------|------|------|
| `--color-primary-*` | 品牌主色系 | `--color-primary`, `--color-primary-light` |
| `--color-surface-*` | 表面层（背景/卡片/浮层） | `--color-surface-base`, `--color-surface-elevated` |
| `--color-text-*` | 文字色 | `--color-text-primary`, `--color-text-secondary` |
| `--color-functional-*` | 功能色（成功/警告/错误） | `--color-functional-success` |
| `--color-player-*` | 播放页专属 | `--color-player-lyric-active` |

**废弃命名（旧版遗留，已统一）**：
- ❌ `--color-bg-base` → ✅ `--color-surface-base`
- ❌ `--color-bg-elevated` → ✅ `--color-surface-elevated`
- ❌ `--color-bg-raised` → ✅ `--color-surface-raised`
- ❌ `--color-bg-card` → ✅ `--color-surface-elevated`
- ❌ `--color-lyric-active` → ✅ `--color-player-lyric-active`

## 深色主题规范

### 为什么要做深色主题？

1. **沉浸感**：深色背景让专辑封面、歌词成为焦点
2. **省电**：OLED 屏幕下黑色像素几乎不耗电
3. **护眼**：低亮度环境下减少眼睛疲劳
4. **一致性**：与 Spotify/Apple Music 等主流音乐 App 一致

### 适配规则

| 元素 | 深色主题规范 |
|------|-------------|
| 背景 | 必须使用深色（#121212 系列），禁止使用浅色背景 |
| 文字 | 主文字 #FFFFFF，次要文字 #B3B3B3，保证 WCAG AA 对比度 |
| 图片 | 专辑封面、歌手头像无需特殊处理，保持原色 |
| 图标 | 使用浅色图标（#FFFFFF / #B3B3B3），禁止深色图标 |
| 按钮 | 主按钮使用 #1DB954，次要按钮使用透明背景 + 白色边框 |
| 卡片 | 背景使用 #181818，可加 1px 边框 rgba(255,255,255,0.1) |

### 动态明暗支持

如需支持亮色主题切换，定义 `--color-surface` 替代固定值：

```css
/* 亮色主题（可选扩展） */
--color-surface: #FFFFFF;
--color-on-surface: #121212;

/* 深色主题（默认） */
--color-surface: #181818;
--color-on-surface: #FFFFFF;
```

## 使用示例

```dart
// Flutter 中的颜色使用
Container(
  color: Color(0xFF121212),  // --color-bg-base
  child: Text(
    '歌曲名称',
    style: TextStyle(color: Color(0xFFFFFFFF)), // --color-text-primary
  ),
)

// 或通过 ThemeData 统一管理
ThemeData(
  brightness: Brightness.dark,
  primaryColor: Color(0xFF1DB954),  // --color-primary
  scaffoldBackgroundColor: Color(0xFF121212),  // --color-bg-base
  cardColor: Color(0xFF181818),  // --color-bg-elevated
)
```
