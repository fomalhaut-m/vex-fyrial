# Typography - 字体规范

## 字体家族选择

### 中文

- **主字体**：PingFang SC（iOS/macOS 系统字体）
- **备选**：Hiragino Sans GB（iOS 中文）、Noto Sans SC（Android）

### 英文/数字

- **主字体**：SF Pro Display（iOS/macOS 系统字体）
- **备选**：SF Pro Text、Roboto（Android）、system-ui

### 等宽字体（时间显示）

- **主字体**：SF Mono（iOS/macOS）
- **备选**：Menlo、monospace

### 字体加载策略

```dart
// Flutter 字体配置
fontFamily: Platform.isIOS ? '.SF Pro Text' : 'Roboto'
```

## 字号规范

### 标题系列

| 样式 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|----------|
| `h1` | 28px | 700 (Bold) | 1.2 | 页面大标题 |
| `h2` | 24px | 600 (SemiBold) | 1.25 | 板块标题（如"推荐歌单"） |
| `h3` | 20px | 600 (SemiBold) | 1.3 | 卡片标题、歌曲名 |
| `h4` | 17px | 600 (SemiBold) | 1.3 | 副标题、小标题 |

### 正文系列

| 样式 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|----------|
| `body1` | 16px | 400 (Regular) | 1.5 | 歌词正文、评论 |
| `body2` | 14px | 400 (Regular) | 1.5 | 描述文字、列表项 |
| `body3` | 12px | 400 (Regular) | 1.4 | 辅助说明 |

### 辅助系列

| 样式 | 字号 | 字重 | 行高 | 使用场景 |
|------|------|------|------|----------|

| `overline` | 10px | 500 (Medium) | 1.2 | 小标签、打赏标识 |
| `button` | 14px | 600 (SemiBold) | 1.0 | 按钮文字 |

## 行高与间距规范

### 基准

- **基础单位**：4px
- **字号基准**：16px（body1）
- **行高基准**：24px（1.5 倍）

### 间距规范

| 名称 | 像素值 | 使用场景 |
|------|--------|----------|
| `space-xs` | 4px | 紧凑间距，如同一行内的元素 |
| `space-sm` | 8px | 小间距，如图标与文字 |
| `space-md` | 12px | 常规间距，如列表项内边距 |
| `space-base` | 16px | 基础间距，如卡片内边距 |
| `space-lg` | 24px | 大间距，如区块之间 |
| `space-xl` | 32px | 特大间距，如页面顶部留白 |
| `space-xxl` | 48px | 最大间距，如底部安全区之外 |

### 垂直节奏

```
页面标题 → 间距 24px → 内容区块
内容区块 → 间距 16px → 内容区块
列表项 → 间距 12px → 列表项
同一列表项内元素 → 间距 8px
```

## 文字省略规范

### 单行省略

```css
overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
```

### 多行省略（最多2行）

```css
display: -webkit-box;
-webkit-line-clamp: 2;
-webkit-box-orient: vertical;
overflow: hidden;
text-overflow: ellipsis;
```

### 省略场景规则

| 场景 | 规则 |
|------|------|
| 歌曲名 | 单行省略，列表场景最多 14 字 |
| 歌手名 | 单行省略，列表场景最多 12 字 |
| 专辑名 | 单行省略 |
| 歌词 | 多行省略（最多2行），播放页完整显示 |

## 字体使用示例

```dart
// Flutter 文字样式
Text(
  '歌曲名称',
  style: TextStyle(
    fontSize: 17,
    fontWeight: FontWeight.w600,
    color: Color(0xFFFFFFFF),
    height: 1.3,
  ),
  maxLines: 1,
  overflow: TextOverflow.ellipsis,
)

// 按钮样式
Text(
  '播放',
  style: TextStyle(
    fontSize: 14,
    fontWeight: FontWeight.w600,
    color: Color(0xFFFFFFFF),
  ),
)
```

## 响应式规则

### 不同屏幕尺寸适配

| 设备 | 基础字号 | 行高调整 | 备注 |
|------|----------|----------|------|
| 手机 (< 375dp) | 14px body | 行高 -2px | 紧凑布局 |
| 手机 (375-414dp) | 16px body | 标准 | 基准 |
| 手机 (> 414dp) | 17px body | 行高 +1px | 宽松布局 |

### 关键尺寸不缩放

- 歌曲名：最小 17px（保证可读性）
- 按钮文字：固定 14px
- TabBar 图标标签：固定 10px
