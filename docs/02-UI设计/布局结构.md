# Layout Structure - 布局结构规范

## 整体架构

### 页面层级

```
┌─────────────────────────────────────┐
│         StatusBar（系统状态栏）       │  沉浸式，全屏内容延伸到状态栏下
├─────────────────────────────────────┤
│                                     │
│         AppBar / TopBar             │  可选，56dp 高，标题/搜索/设置
│                                     │
├─────────────────────────────────────┤
│                                     │
│                                     │
│           Content Area              │  内容区域，高度随设备动态计算
│         （SafeArea 内）              │
│                                     │
│                                     │
├─────────────────────────────────────┤
│         MiniPlayer（迷你播放器）      │  固定 64dp 高，播放时显示
├─────────────────────────────────────┤
│         TabBar（底部导航栏）          │  固定 56dp 高，4 个 Tab
└─────────────────────────────────────┘
```

### 安全区处理

```dart
// Flutter SafeArea 使用规范
SafeArea(
  top: true,   // 顶部状态栏
  bottom: true, // 底部圆角home键区域 + TabBar + MiniPlayer
  child: content,
)
```

### 沉浸式布局

```dart
// 全屏沉浸式（播放页）
MediaQuery.of(context).padding.top  // 获取状态栏高度
// 内容延伸到状态栏下，使用 SystemUiOverlayStyle 设置状态栏图标颜色

// 常规布局
Scaffold(
  appBar: AppBar(...),  // 系统自动处理 SafeArea
  bottomNavigationBar: TabBar(...),  // 系统自动处理底部安全区
)
```

## TabBar 底部导航栏规范（v3）

> ⚠️ v3 修订：TabBar 更新为 4 Tab：播放器 | 播放列表 | 统计 | 设置

**4 个 Tab**：播放器 | 播放列表 | 统计 | 设置

| Tab | 图标 | 文字 | 说明 |
|------|------|------|------|
| Tab 1 | music_note / play_circle | 播放器 | 当前播放内容（封面/歌词/播控） |
| Tab 2 | queue_music | 播放列表 | 本地音乐 + 歌单 |
| Tab 3 | bar_chart | 统计 | 播放统计（按日/周/月 + 分类） |
| Tab 4 | settings | 设置 | OSS配置 + 播放设置 + 关于 |

### Tab 配置

```dart
bottomNavigationBar: BottomNavigationBar(
  type: BottomNavigationBarType.fixed,  // 固定位置，不随内容滚动
  backgroundColor: Color(0xFF181818),
  selectedItemColor: Color(0xFF1DB954),
  unselectedItemColor: Color(0xFFB3B3B3),
  currentIndex: currentTabIndex,
  onTap: (index) => switchTab(index),
  items: [
    BottomNavigationBarItem(icon: Icon(Icons.music_note), label: '播放器'),
    BottomNavigationBarItem(icon: Icon(Icons.queue_music), label: '播放列表'),
    BottomNavigationBarItem(icon: Icon(Icons.bar_chart), label: '统计'),
    BottomNavigationBarItem(icon: Icon(Icons.settings), label: '设置'),
  ],
)
```

## 顶部导航规范

### AppBar 高度

- **标准 AppBar**：56dp
- **无标题 AppBar**：48dp
- **搜索框 AppBar**：56dp（含搜索框）

### AppBar 内容

```dart
AppBar(
  title: Text('标题'),  // 居中，h3 样式
  centerTitle: true,
  backgroundColor: Color(0xFF121212),  // 透明或背景色
  elevation: 0,  // 无阴影
  leading: IconButton(...),  // 左侧返回/菜单
  actions: [IconButton(...)],  // 右侧操作按钮
)
```

### 返回按钮位置

- 左侧 16dp 处
- 图标大小：24dp × 24dp
- 点击热区：48dp × 48dp

## 迷你播放器（MiniPlayer）

### 尺寸规范

| 属性 | 值 |
|------|---|
| 高度 | 64dp |
| 封面尺寸 | 48dp × 48dp |
| 封面圆角 | 4dp |
| 左侧边距 | 12dp |
| 封面与文字间距 | 12dp |

### MiniPlayer 结构

```
┌──────────────────────────────────────────────────────────┐
│ [封面] 歌曲名称                      [播放/暂停] [下一首] │
│         歌手名 · 02:35                    icon  icon     │
└──────────────────────────────────────────────────────────┘
```

### 交互规范

| 交互 | 行为 |
|------|------|
| 点击封面/歌曲名 | 打开全屏播放页 |
| 点击播放/暂停按钮 | 切换播放状态 |
| 点击下一首按钮 | 切到下一首 |
| 左右滑动手势 | 切歌（可选） |
| 点击 MiniPlayer 任意区域 | 打开全屏播放页 |

### 进度指示

- MiniPlayer 顶部显示 2dp 高度进度条
- 颜色：#1DB954
- 背景：rgba(255,255,255,0.1)

## 页面跳转动画规范

### 基础参数

| 属性 | 值 |
|------|---|
| 动画时长 | 300ms |
| 缓动函数 | Curves.easeInOut |
| 方向 | 从右滑入（默认）/ 从下滑入（Modal） |

### 页面切换动画

```dart
// 标准页面跳转
Navigator.push(
  context,
  PageRouteBuilder(
    transitionDuration: Duration(milliseconds: 300),
    pageBuilder: (context, animation, secondaryAnimation) => TargetPage(),
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      return SlideTransition(
        position: Tween<Offset>(
          begin: Offset(1.0, 0.0),
          end: Offset.zero,
        ).animate(CurvedAnimation(
          parent: animation,
          curve: Curves.easeInOut,
        )),
        child: child,
      );
    },
  ),
);

// 返回
Navigator.pop(context);  // 300ms 从左滑出
```

### 底部弹出（Modal 形式）

```dart
// 全屏播放页 / 歌单详情页
showModalBottomSheet(
  context: context,
  isScrollControlled: true,  // 内容可滚动
  backgroundColor: Colors.transparent,  // 透明背景
  transitionAnimationController: AnimationController(
    duration: Duration(milliseconds: 300),
    vsync: this,
  ),
);

// 或使用自定义动画
Navigator.push(
  context,
  PageRouteBuilder(
    fullscreenDialog: true,  // iOS 形式从底部弹出
    transitionDuration: Duration(milliseconds: 300),
    pageBuilder: (context, animation, secondaryAnimation) => PlayerPage(),
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      return SlideTransition(
        position: Tween<Offset>(
          begin: Offset(0.0, 1.0),
          end: Offset.zero,
        ).animate(CurvedAnimation(
          parent: animation,
          curve: Curves.easeInOut,
        )),
        child: child,
      );
    },
  ),
);
```

### 列表页 → 详情页

```
推荐列表页 → 歌单详情页（从右滑入，300ms）
歌单详情页 → 歌曲播放（全屏弹出，从下滑入）
```

### 特殊页面动画

| 场景 | 动画 |
|------|------|
| Tab 切换 | 无动画，Instant switch |
| 全屏播放页 | 从底部滑入，iOS 风格 |
| 搜索页 | 从右滑入（覆盖而非替换） |
| 设置页 | 从右滑入 |
| 分享/弹窗 | 淡入 + 轻微缩放 |

## 响应式规则

### 手机竖屏（主要适配）

- 标准布局，如上所述

### 手机横屏

| 区域 | 调整 |
|------|------|
| TabBar | 移到左侧（56dp 宽），图标垂直排列 |
| 内容区域 | 宽高比变化，网格列数可能增加 |
| MiniPlayer | 高度减为 56dp |
| 播放页 | 左 60% 封面，右 40% 歌词/控制 |

### 平板（适配优先级：低）

- 双列布局，左侧列表，右侧详情
- TabBar 变为左侧边栏

## 布局检查清单

- [ ] 状态栏图标是否为浅色（白色/灰色）
- [ ] 内容是否在 SafeArea 内
- [ ] 底部是否有足够空间给 TabBar + Home 键区域
- [ ] MiniPlayer 是否覆盖内容底部，而非被内容撑开
- [ ] 页面切换动画是否为 300ms ease-in-out
- [ ] 点击热区是否足够大（至少 48dp × 48dp）
