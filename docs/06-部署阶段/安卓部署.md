## 3. 签名配置

### 3.1 签名密钥

- **Debug 签名**：使用 Android Studio 自动生成的 debug.keystore
- **Release 签名**：使用独立密钥库，需配置 `key.properties`

```properties
# key.properties
storePassword=<STORE_PASSWORD>
keyPassword=<KEY_PASSWORD>
keyAlias=vexfy
storeFile=/path/to/vexfy.jks
```

### 3.2 build.gradle 配置

```groovy
def keystorePropertiesFile = rootProject.file("key.properties")
def keystoreProperties = new Properties()
keystoreProperties.load(new FileInputStream(keystorePropertiesFile))

android {
    signingConfigs {
        release {
            keyAlias keystoreProperties['keyAlias']
            keyPassword keystoreProperties['keyPassword']
            storeFile file(keystoreProperties['storeFile'])
            storePassword keystoreProperties['storePassword']
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
            minifyEnabled true
            shrinkResources true
            proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
        }
    }
}
```

```proguard
# Aliyun OSS SDK

```

## 5. 构建渠道

### 5.1 Debug 构建

```bash
flutter build apk --debug
```

### 5.2 Release 构建

```bash
flutter build apk --release
```

### 5.3 Split APK（按 ABI）

```bash
flutter build apk --release --split-per-abi
```

生成：
- `build/app/outputs/apk/release/app-arm64-v8a-release.apk`
- `build/app/outputs/apk/release/app-armeabi-v7a-release.apk`
- `build/app/outputs/apk/release/app-x86_64-release.apk`

## 6. Google Play Store 上架

### 6.1 必填材料

| 材料 | 说明 |
|------|------|
| 应用签名证书 | SHA-1 指纹需在 Play Console 注册 |
| 应用图标 | 512x512 PNG |
| 功能截图 | 手机截图 4-8 张 |
| 隐私政策 URL | 必须托管在 HTTPS 页面 |
| 内容分级问卷 | Play Console 在线填写 |

### 6.2 App Bundle

```bash
flutter build appbundle --release
```

生成文件：`build/app/outputs/bundle/release/app-release.aab`

### 6.3 关键清单项

```xml
<!-- AndroidManifest.xml -->
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.FOREGROUND_SERVICE" />
<uses-permission android:name="android.permission.FOREGROUND_SERVICE_MEDIA_PLAYBACK" />
<uses-permission android:name="android.permission.WAKE_LOCK" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" android:maxSdkVersion="32" />
<uses-permission android:name="android.permission.READ_MEDIA_AUDIO" />
```
