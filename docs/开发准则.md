## 二、代码质量要求

### 2.1 代码可读性（最重要）

**代码必须像教程一样清晰，因为 Luke 会通过读代码学习 Flutter：**

- **中文注释**：每个关键逻辑必须有中文注释
- **解释思路**：不只说"做了什么"，还要说"为什么这么做"
- **命名清晰**：变量/函数/类名一看就懂，不用猜
- **目录清晰**：目录结构就是代码的目录，一目了然

### 2.2 项目结构

```
lib/
├── main.dart              # 入口，注释说明整个App结构
├── app/                   # 应用层
│   ├── modules/           # 功能模块（按页面/功能划分）
│   │   ├── player/         # Tab1 播放器模块
│   │   ├── playlist/       # Tab2 播放列表模块
│   │   ├── stats/          # Tab3 统计模块
│   │   └── settings/        # Tab4 设置模块
│   ├── routes/             # 路由配置（注释说明路由结构）
│   └── core/               # 核心常量、主题、工具
├── data/                  # 数据层
│   ├── models/             # 数据模型（注释说明每个字段）
│   ├── providers/           # 数据源（本地/远程）
│   └── repositories/        # 数据仓库（封装数据操作）
├── services/              # 公共服务
│   ├── player_service.dart  # 播放器服务
│   ├── storage_service.dart # 本地存储服务
│   └── oss_sync_service.dart# OSS同步服务
└── widgets/                # 通用组件
    ├── mini_player.dart    # 全局迷你播放器
    └── ...
```

### 2.3 开发原则

1. **先理解再写代码**
   - 写代码前先看完对应的 PRD + UI 设计
   - 理解为什么要做这个功能
   - 确认设计方案再动手

2. **代码自审清单**（每次提交前自己检查）
   - [ ] 代码逻辑清晰吗？
   - [ ] 注释够用吗？能让人看懂吗？
   - [ ] 命名清晰吗？
   - [ ] 有没有性能问题？
   - [ ] 功能实现和 UI 设计一致吗？

3. **反复思考**
   - 写完问自己："这样合理吗？"
   - 考虑可维护性、可扩展性
   - 不只写能跑的代码，要写好代码

## 三、P0 开发任务（首阶段）

根据 PRD v3，P0 优先级功能：

### P0-1：本地音乐文件扫描
- 指定目录递归扫描
- 解析音频文件元数据（ID3标签）
- 存入本地数据库

### P0-2：音频播放
- 播放/暂停/停止
- 上一首/下一首
- 进度拖动
- 播放模式（列表循环/单曲循环/随机）
- **必须流畅不卡**

### P0-3：后台播放 + 通知栏控制
- App 切后台继续播放
- 通知栏显示（封面/歌名/播控）
- 锁屏控制

### P0-4：内存优化
- 内存占用 < 150MB
- 不杀后台

## 五、注释规范

### 必须注释的内容

```dart
// ✅ 好注释：解释为什么这么做
// 使用 GetX 的原因：需要全局状态共享，跨页面同步播放状态
class PlayerController extends GetxController {

  // ✅ 好注释：解释功能
  // 扫描指定目录，返回所有音频文件路径列表
  // 递归扫描子目录，支持 MP3/AAC/FLAC/WAV 格式
  Future<List<String>> scanDirectory(String path) async {
    // ...
  }

  // ✅ 好注释：解释关键逻辑
  // 计算播放进度百分比，用于更新进度条
  // 这里用 Duration.inMilliseconds 而不用 total.inSeconds
  // 是为了保证进度条更新平滑，不卡顿
  double get progressPercent {
    if (total.inMilliseconds == 0) return 0;
    return current.inMilliseconds / total.inMilliseconds;
  }
}
```

### 禁止的注释

```dart
// ❌ 废话注释：说了等于没说
// 这是一个播放暂停功能
void playPause() { ... }

// ❌ 模糊注释：不解释原因
// 设置播放模式
void setPlayMode(mode) { ... }

// ❌ 拼音注释：不用拼音
// 播放音乐
void boFang() { ... }
```
